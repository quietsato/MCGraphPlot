# MCGraphPlot

A bukkit plugin that plots graphs in Minecraft worlds.

## Demo

![demo](./docs/demo.gif)

## Usage Examples

Plot a graph of `x**2+y**2+z**2-1=0`.

```
/gp x**2+y**2+z**2-1
```

You can use [SymPy](https://github.com/sympy/sympy) functions to write equations.
For example,

```
/gp sin(x)-y
```

## Prerequisites

- Python3
- SymPy
